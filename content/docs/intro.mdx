## Overview

ednesdayW provides production-ready React components that are:
- **Accessible** - Built on Radix UI primitives with WCAG compliance
- **Customizable** - Styled with Tailwind CSS for easy theming
- **Type-safe** - Written in TypeScript with full type definitions

## Tech Stack

- React 19
- Next.js 15
- Radix UI
- Tailwind CSS v4
- TypeScript
